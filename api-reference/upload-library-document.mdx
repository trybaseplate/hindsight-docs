Upload competitive intelligence documents, assets, and battlecards to your library.

<Accordion title="Copy for AI context">
```text

Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

Request body:
{
  "file_name": "string (required)",
  "file_url": "string (uri) — URL to fetch file from",
  "file_base64": "string — base64-encoded file content (alternative to file_url)",
  "content_type": "string (required) — MIME type, e.g. application/pdf",
  "type": "asset | intel (required)",
  "competitor_id": "string — Hindsight competitor ID",
```

</Accordion>

## Document Types

| Type | Description |
|------|-------------|
| `asset` | Sales enablement content like battlecards, one-pagers, guides |
| `intel` | Competitive intelligence gathered from external sources |

## Source Options

The `source` field determines the icon displayed in the Hindsight UI:

| Source | Description |
|--------|-------------|
| `drive` | Google Drive |
| `notion` | Notion |
| `confluence` | Confluence |
| `onedrive` | OneDrive |
| `sharepoint` | SharePoint |
| `vanta` | Vanta |
| `url` | Web URL |
| `zapier` | Zapier automation |
| `api` | API upload |

<RequestExample>
```bash cURL
curl -X POST https://api.usehindsight.com/api/v1/documents/library \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "file_name": "Competitor X Pricing Guide.pdf",
    "file_url": "https://example.com/document.pdf",
    "content_type": "application/pdf",
    "type": "asset",
    "competitor_name": "Competitor X",
    "source": "drive"
  }'
```

```javascript Node.js
const response = await fetch('https://api.usehindsight.com/api/v1/documents/library', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    file_name: 'Competitor X Pricing Guide.pdf',
    file_url: 'https://example.com/document.pdf',
    content_type: 'application/pdf',
    type: 'asset',
    competitor_name: 'Competitor X',
    source: 'drive'
  })
});

const data = await response.json();
console.log(`Document uploaded: ${data.document.id}`);
```

```python Python
import requests

response = requests.post(
    'https://api.usehindsight.com/api/v1/documents/library',
    headers={
        'Authorization': f'Bearer {API_KEY}',
        'Content-Type': 'application/json'
    },
    json={
        'file_name': 'Competitor X Pricing Guide.pdf',
        'file_url': 'https://example.com/document.pdf',
        'content_type': 'application/pdf',
        'type': 'asset',
        'competitor_name': 'Competitor X',
        'source': 'drive'
    }
)

data = response.json()
print(f"Document uploaded: {data['document']['id']}")
```

<RequestExample>
```bash cURL
curl -X POST https://api.hindsight.com/api/v1/responses \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [
      {
        "role": "user",
        "content": "What security objections have we encountered with Enterprise customers?"
      }
    ],
    "org_id": "your_org_id",
    "stream": false
  }'
```

```javascript Node.js
const response = await fetch('https://api.hindsight.com/api/v1/responses', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    messages: [{ role: 'user', content: 'What security objections have we encountered?' }],
    org_id: 'your_org_id',
    stream: false
  })
});

const data = await response.json();
console.log(data.message.content);
```

```python Python
import requests

response = requests.post(
    'https://api.hindsight.com/api/v1/responses',
    headers={
        'Authorization': f'Bearer {API_KEY}',
        'Content-Type': 'application/json'
    },
    json={
        'messages': [{'role': 'user', 'content': 'What security objections have we encountered?'}],
        'org_id': 'your_org_id',
        'stream': False
    }
)

data = response.json()
print(data['message']['content'])
```
